<h2>Topics Covered (Indicative)</h2>

<p>
The course covers the core topics of introductory and intermediate business statistics, including:
</p>

<ul>
  <li>Descriptive Statistics</li>
  <li>Data Visualization and Graphical Methods</li>
  <li>Probability Concepts</li>
  <li>Discrete Random Variables</li>
  <li>Continuous Random Variables</li>
  <li>Sampling and Sampling Distributions</li>
  <li>Confidence Intervals</li>
  <li>Hypothesis Testing</li>
  <li>Statistical Inference</li>
  <li>Correlation and Simple Linear Regression</li>
  <li>Multiple Regression and Model Building</li>
  <li>Chi-Square Tests</li>
</ul>
